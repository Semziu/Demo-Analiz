Demo Analiz

Pobierz program do analizy dem, porownan graczy i bety Live Coach. Strona jest przygotowana tak, zeby latwo udostepniac program, a aktualizacje szly przez GitHub Releases.

Pobierz .exe Setup GitHub Releases
Aktualna wersja
1.5.0 Windows

Program potrafi teraz sprawdzac najnowszy GitHub Release i pobierac aktualizacje bez recznego linkowania pliku .exe.

Co potrafi program

  • Analiza pojedynczego dema i serii kilku dem
  • Biblioteka dem trzymana lokalnie w programie
  • Porownanie Twojego dema z demem lepszego gracza
  • Beta Live Coach z historia sesji
  • System auto-aktualizacji przez GitHub Releases

Jak wrzucac aktualizacje

  1. Zmien numer APP_VERSION w programie.
  2. Wypchnij zmiany do repozytorium GitHub.
  3. Utworz tag w stylu v1.5.1.
  4. GitHub Actions zbuduje nowe .exe i wrzuci je do Releases.
