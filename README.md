# Demo Analiz

Desktopowy analizator dem CS2 z raportem trenerskim, analiza wielu dem, porownaniem dwoch graczy, biblioteka dem i beta `Live Coach`.

## Najwazniejsze funkcje

- analiza pojedynczego dema
- analiza kilku dem naraz
- porownanie `Ty vs lepszy gracz`
- biblioteka dem
- `Live Coach Beta`
- aktualizacje aplikacji przez `GitHub Releases`

## Aktualizacje

Aplikacja domyslnie sprawdza release dla repo:

- `Semziu/Demo-Analiz`

Po opublikowaniu nowszego releasu z plikiem `DemoAnaliz.exe` uzytkownik moze zaktualizowac program z poziomu aplikacji.

## Pliki

- glowny kod: `Test.py`
- gotowy build: `dist/DemoAnaliz.exe`
- workflow release: `.github/workflows/release.yml`
- instrukcja: `GITHUB_RELEASES_SETUP.md`
